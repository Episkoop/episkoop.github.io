Создание HTML страницы

В время прохождения производственной практики в компании ООО ТД "МСТ", мой наставник предаставил мне задачу создать веб страницу на основе HTML. Первое что нам необходимо сделать - открыть программу "Sublime Text", в которой мы будем писать наш сайт. Создаем новую страничку нажав на кнопку "File" и выбрав "New" в верхней панели. Теперь создаём ещё одну страничку, но на этот раз с расширением CSS, для того, чтобы у нашего сайта были стили. Добавляем меню навигации для следующих страничек, ищем для неё CSS стиль или создаём его сами. Добавляем новые классы в CSS для того, чтобы сайт смотрелся лучше. Я использовал контейнеры HEADER для создания "шапки" сайта; UL, LI, A и BUTTON для создания кнопки в меню сайта; DIV для блоков с текстом. Прописываем фон сайта и также создаем фон для меню, задаем размер меню в пикселях, убираем подчеркивание ссылки и маркер списка в кнопке сайта, и делаем так, что бы кнопки выстраивались в одну линию. Корректируем внешний вид кнопки "Главная" в меню скругляя углы, меняем цвет, и также задаем нужный размер шрифта. Приступаем к вёрстке нашей странице. Создаем контейнер DIV в него прописываем текст, на всё это накидываем стили добавляем картинки с помощью команды IMG создаём под картинки стиль и применяем его к ним, и вот у нас получилась простая HTML страничка.
